# 01. Scientific Scope and Status

## 문제 정의

대상은 다음 세 부류다.

1. 표준모형 gauge sector: \(g_1,g_2,g_3\), \(\hatlpha_{em}(M_Z)\), \(\sin^2\hat	heta_W(M_Z)\), \(lpha_s(M_Z)\)
2. flavour/neutrino sector: Yukawa mass ratios, CKM, PMNS, neutrino ordering/splittings
3. ΛCDM/cosmology sector: \(\Lambda\), \(\Omega_c h^2\), \(N_{eff}\), 기타 최소 6개 ΛCDM 파라미터

## 엄격한 구분

| 등급 | 의미 | 현재 상태 |
|---|---|---|
| measured input | 측정값을 경계조건으로 사용 | SM+GR control |
| conditional derivation | gauge group/field content가 주어지면 계산 | beta function 등 |
| candidate prediction | 후보 이론이 산출하지만 아직 공인 아님 | ISDLC–TCPS gauge sector |
| post-selected hit | target을 보고 찾은 수식 | autodiscovery 후보 대부분 |
| blind prediction | freeze 전에 값 미공개 holdout 예측 | internal Cabibbo-clock만 통과 |
| external blind success | 독립 tranche에서 통과 | 없음 |

## 최종 과학 판정

현재 엔진은 “이론 발견기”가 아니라 **이론 검증·반증·후보 생성 보조기**로 보는 것이 맞다. 숫자를 잘 맞히는 후보는 여럿 생성되지만, 외부 holdout 실패 또는 미예측 항목 때문에 완전한 제1원리 이론은 아직 없다.
