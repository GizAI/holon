# Holon Research Archive

This repository is split into three top-level research tracks.

## `ocean/`

BHF-OCEAN final v0.2 canonical package, including compact claims docs,
final audit engine, frozen outputs, source inventory, and manuscript.

Start with:

- `ocean/README_ko.md`
- `ocean/docs/FINAL_KO_COMPACT.md`
- `ocean/manuscript/bhf_ocean_manuscript_v0_2.md`

## `fpphysics/`

Consolidated first-principles physics engine for strict scoring, candidate
discovery, blind-prediction packet freezing, and external holdout audits.

Start with:

- `fpphysics/README.md`
- `fpphysics/docs/INDEX_ko.md`
- `fpphysics/artifacts/results/consolidated_summary.json`

notes, and earlier simulation modules.

Start with:

- `holon/README.md`
